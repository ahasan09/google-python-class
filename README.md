# Google Python Class

Exercises and solutions from [Google's Python Class](https://developers.google.com/edu/python), a free Python course for people with some programming experience.

## Prerequisites

- [Python](https://www.python.org/) 3.x

## Exercises

| Folder / File | Topic |
|---------------|-------|
| `basic/` | Python basics — strings, lists, sorting |
| `babynames/` | Parsing HTML, extracting data |
| `copyspecial/` | Working with files and directories |
| `logpuzzle/` | URL parsing, image reconstruction |
| `hello.py` | Hello World starter script |

## Running the Exercises

```bash
# Run any script directly
python3 hello.py
python3 basic/string1.py

# Run a specific exercise module
python3 babynames/babynames.py babynames/baby1990.html
python3 copyspecial/copyspecial.py
python3 logpuzzle/logpuzzle.py logpuzzle/animal_code.google.com
```
