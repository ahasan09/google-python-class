# Google Python Class

Python exercises from Google's Python Class covering strings, lists, HTML parsing, file operations, and log puzzles.

## Tech Stack
- Python 3 (no external dependencies)

## Project Structure
```
google-python-class/
├── basic/
│   ├── string1.py / string2.py
│   └── list1.py / list2.py
├── babynames/         # HTML parsing exercise
├── copyspecial/       # File copying exercise
└── logpuzzle/         # Log parsing puzzle
```
Each exercise folder has a `solution/` subfolder with the reference solution.

## Development
```bash
# Run any exercise directly
python <exercise>.py
```
No installation or virtual environment required.

## Key Notes
- Pure Python standard library — no pip installs needed.
- Each exercise has starter code with TODO markers and a `solution/` folder for reference.
